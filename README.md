# Projeto Mobile – Liderança Empática (Alimentos) :seedling:

## 📌 Objetivo
Este é o repositório do **AlimepatIA**, um aplicativo focado na liderança empática aplicada à gestão de alimentos. Ele permite acompanhar e distribuir melhor os produtos como arroz, feijão e itens básicos, bem como monitorar o bem-estar e o desempenho dos membros da equipe.

## 🛠️ Tecnologias Utilizadas
- **Kotlin** & **Android Studio**
- **Arquitetura MVVM** (Activities/Fragments, ViewModels, etc)
- **Jetpack Navigation** e **Material Design**
- **GitHub Pages** (Para esta Documentação)
